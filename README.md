## 题目一：HTML/CSS

### 测试要求

请仔细查看设计稿，按照设计稿要求尽力完成页面，图片素材中包含了页面所需的图标、背景图。

### 交互规则

1. 点击头部导航栏4个文字标题会出现下拉菜单
2. 鼠标移动到“see more”上面会出现气泡框内容，气泡框出现5秒（5秒之内不会消失）
3. 鼠标移动到“气泡框”上持续显示
4. 鼠标移出“气泡框”则3秒淡出消失不再显示
